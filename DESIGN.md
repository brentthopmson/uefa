# Design Specification: UEFA Premium

This document defines the elite visual and behavioral standards for the UEFA ticketing platform, aligning with the official tournament branding.

## 1. Global Components

### 1.1 Bottom Navigation Bar (Mobile)
- **Background**: Solid `#FFFFFF` (White).
- **Active State**: Icon and label color `#001C4B` (UEFA Dark Blue).
- **Inactive State**: Icon and label color `#99A1B3` (Subtle Gray).
- **Items**:
  - **Discover**: `faSearch` | "Discover"
  - **Favorites**: `faHeart` | "Favorites"
  - **My Tickets**: `faTicketAlt` | "My Tickets"
  - **My Account**: `faUserCircle` | "My Account"

### 1.2 Fixed Header
- **Background**: `#FFFFFF` (White).
- **Title**: Centered, Bold, UEFA Dark Blue.
- **Controls**: Side menu toggle (left) and Profile link (right).

## 2. Page: My Purchases

### 2.1 Match Grids
- Matches are displayed in a clean, vertical list or grid.
- **Teams**: Displayed with official logos and names.
- **Tournament Branding**: Subtle gradient backgrounds or official iconography.

### 2.2 Match Details
- Clear separation between Match Info (Teams, Date, Venue) and Ticket Info (Seat, Row, Block).
- **Maps**: Integrated stadium maps for seat orientation.

## 3. Colors & Typography

### 3.1 Color Palette
- **UEFA Dark Blue**: `#001C4B`
- **UEFA Cyan**: `#00F0FF`
- **Background**: White or very light gray.

### 3.2 Typography
- **Primary Font**: Premium sans-serif (e.g., Inter or Outfit).
- **Titles**: Bold, Dark Blue, large font size.
- **Metadata**: Regular, Gray, small font size.
